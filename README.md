# VividCanvas 🎨
A high-performance, real-time collaborative whiteboard built with Flutter and Firebase.

## 🚀 Features (Planned)
- **Infinite Canvas:** Pan and zoom across an endless workspace.
- **Real-time Collaboration:** See other users' cursors and edits instantly.
- **Miro-style Tools:** Sticky notes, shapes, and freehand drawing.
- **Cross-Platform:** Optimized for Web and Tablet.

## 🛠️ Tech Stack
- **Framework:** Flutter (Web/Mobile)
- **State Management:** Riverpod
- **Backend:** Firebase (Firestore & Auth)
- **Drawing:** CustomPainter API

## 📂 Architecture
This project follows **Clean Architecture** principles to ensure scalability and testability.
